# Shopping Rules

Check this file before creating any Rohlik shopping list from a recipe.

## Never Buy (Make at Home)

- Pickled red onions - Martin always makes these at home
- Pickled vegetables in general - check before adding

## Spice Mixes

- When a recipe calls for a pre-made spice mix (gyros, taco, fajita, etc.), first suggest the individual spice components so Martin can make it from what he has at home
- Only add the pre-made mix if Martin explicitly wants it
- Example: "taco seasoning" -> suggest: cumin, chili powder, paprika, garlic powder, onion powder, oregano, salt

## Pantry Staples (Skip Unless Asked)

- Olive oil
- Salt, pepper
- Basic dried spices (paprika, cumin, oregano, chili)
- Garlic (usually has at home)
